Case Studies
CASE 01

Failure Analysis — 테스트 칩 HTOL 신뢰성 불량

JEDEC · X-ray · EMMI · C-SAM · Package

FCBGA 테스트 칩 HTOL 시험 중 Readout이 누적될수록 단락성 불량이 늘어나는 구조적 신뢰성 이슈가 발생. 데이터 기반 배제법으로 설계와 보드를 순차 배제하고, 물리분석으로 패키지 Underfill 기인 Solder Extrusion을 Root Cause로 규명했습니다.

HTOL Fail 분류 → 설계 Review 배제 → 보드 Review 배제 → X-ray 이상점 검출 → EMMI Hot Spot → Decap → 물리분석 → Solder Extrusion → 인접 Ball Short

소재 물성(CTE·Modulus·Tg) 비교로 Underfill 대체재를 선정하고, C-SAM 사전 검증과 소재 Qualification을 거쳐 동일 조건 후속 시험 Pass로 개선을 검증 — 재발 없이 종결했습니다.

Root Cause 규명소재 물성 기반 의사결정재검증 Pass · 재발 0

CASE 02

LabVIEW 계측 자동화 — PVT Corner 측정

LabVIEW · FPGA · VISA · FSM

측정 전담팀 없이 수작업으로 운용되던 특성 측정을 직접 제안·도입한 자동화로 해결. 온도조절기·오실로스코프·FPGA·Power Supply를 하나의 SW로 통합 제어하고, FSM 기반 온도·전원 제어 로직으로 PVT Corner 전 구간을 자동 순차 측정합니다.

WRITE_VECTOR → 온도·전원 세팅 → 측정 → 기록 → 다음 Corner (자동 반복 · 리포트 생성)
측정 시간 3배 이상 단축재현성·추적성 확보인적 오류 제거

CASE 03

QMS — Quality Control Hub

Python · Rust · 로컬 LLM/RAG · Podman

이슈·스펙·문서가 흩어져 현황 파악이 어려웠던 개발부서를 위해, 클라우드를 쓸 수 없는 폐쇄망·root 권한 없는 서버라는 제약 위에 인프라부터 AI 기능까지 풀스택 품질 플랫폼을 구축했습니다.

Podman rootless 컨테이너로 전 개발부서 대상 서비스를 안정 운영하고, llama.cpp 기반 로컬 LLM과 사내 문서 벡터 DB를 결합한 출처 기반 RAG 검색, Document Studio, APQP 자동화를 구현했습니다. 성능이 필요한 Register 엔진 백엔드는 Python에서 Rust로 마이그레이션했습니다.

프로젝트·이슈·스펙 통합로컬 LLM + RAGAPQP 자동화Rust 마이그레이션

CASE 04

DQV Sign-off Hub — IP 출하 정합성 검증

Flask · Python · Rule Deck

IP를 고객에 release할 때 문서·Front-end·Back-end 간 정합성 오류가 잦았던 문제를, 출하 전 자동으로 잡아내는 사인오프 리뷰 툴로 해결했습니다. 파운드리 표준 필수 룰을 도메인별 checker(Verilog·LEF·Liberty·DRC·LVS·datasheet 등)에 매핑한 Traceability Matrix를 구축하고, IP 등록 → 검증 Job 실행 → 웹 리포트로 이어지는 파이프라인을 구현했습니다.

ISO 26262 기능안전 인증 체계 구축

FMEDA · DFA · Safety Case/GSN · Jira

기능안전 인증 체계를 초기부터 구축했습니다. ISO 26262 Part 2/5/8/9에 Part 11 반도체 가이드를 적용하고, Permanent/Transient FIT를 분리한 FMEDA, DFA, GSN 기반 Safety Case, Tool Qualification, Jira 기반 요구사항 추적성을 수립했으며 외부 인증기관 심사를 영어로 대응했습니다.
